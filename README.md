# Parrot - A yt-dlp frontend

# Planned Features
- [ ] Download input URL
- [ ] Search bar
- [ ] Playlist support (Download Queue ?)
- [ ] Download history
- [ ] Download progress
- [ ] Preview of the video (thumbnail, title, etc.)
- [ ] Presets for download options (format, quality, etc.)
- [ ] Settings (output directory, etc.)
- [ ] Yt-dlp checker (check if yt-dlp is installed and up to date else prompt user to install/update it)
- [ ] add meta data to the downloaded file (title, thumbnail, etc.)

# Tech stack
- (frontend) Metaframework : SvelteKit
- Packet manager : bun
- web server : Vite
- Tests : Vitest
- Framework : Tauri
- UI-library : Shadcn UI
- CSS : Tailwind CSS

# Dependencies
- [yt-dlp](https://github.com/yt-dlp/yt-dlp)
